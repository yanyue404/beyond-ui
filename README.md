# beyond-ui

## 快速开始

### 1.安装组件库

```bash
npm i beyond-ui
```

### 2.引用组件库

```javascript
// 全部引入
import BeyondUI from "beyond-ui";
import "beyond-ui/lib/style/index.css";
Vue.use(BeyondUI);

// 按需引用
import { Demo } from "beyond-ui";
import "beyond-ui/lib/style/demo.css";
Vue.use(Demo);
```

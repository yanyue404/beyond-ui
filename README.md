# Beyond-ui

## 快速开始

### 1.安装组件库

```bash
npm i beyond-ui
```

### 2.引用组件库

```javascript
// 全部引入
import "beyond-ui/dist/css/index.css";
import BeyondUI from "beyond-ui";
Vue.use(BeyondUI);

// 按需引用
import "beyond-ui/dist/css/demo.css";
import { Demo } from "beyond-ui";
Vue.use(Demo);
```
